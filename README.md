# FOCUS ZONE

 - FOCUS ZONE is a personal task management app used to store, organize and prioritize your tasks.

 - Visit here: https://praveen-task-management-app.netlify.app

# Tech Stack

### Front End Tool

- React.JS
-	Tailwind CSS
-	Formik Validation Tool
-	React Router Client side routing Tool
-	Redux and Context API
-	Vite
-	Flowbite

### Back End Tool

- Node.js
-	Express.js
-	API Handle Test
-	MongoDB
-	Authentication
-	Authorisation
-	Bcrypt

# Dependencies

### The following dependencies are required to run this project:

 - bcrypt
 - cookie-parser
 - cors
 - dotenv
 - express
 - jsonwebtoken
 - jwt-decode
 - mongoose
 - morgan
 - multer
